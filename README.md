# House Price Prediction using Machine Learning

# Objective
Predict house prices using machine learning regression models based on property features.

# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

# Features Used
- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Furnishing Status
- Main Road Access

# Machine Learning Models
- Linear Regression
- Gradient Boosting Regressor

# Evaluation Metrics
- MAE
- RMSE
- R² Score
  
# Final Code Flow
- Import Libraries
- Load Dataset
- Dataset Info
- Encode Categorical Features
- Define X and y
- Feature Scaling
- Train-Test Split
- Train Gradient Boosting Model
- Predictions
- Evaluate Model
- Visualization

# Expected Output

You’ll see:

- Dataset preview
- Dataset info
- Missing values
- Model trained message
- MAE
- RMSE
- R² Score
- Graph of Actual vs Predicted Prices

# Output
The project predicts house prices and visualizes actual vs predicted prices.
